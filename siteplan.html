Development notes

  • Canonical site width: 840px (main layout controlled in `styles/styles.css`).
  • Use CSS variables in `:root` for colors and typography when possible.
  • Keep page-specific CSS small and load it after the global stylesheet.
  • Prefer semantic HTML (use <table> only for tabular data, forms should use fieldsets and labels).
